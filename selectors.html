Selectors in style sheet

Selectors are used to indicate the HTML element to which we want to apply the style. A selector can be an aelement name, attirubute, class, id, etc.

Id selector

When we use id selector tyhe style will be applied to the element that have the specified id.

Id attribute

Id attribute gives a unique id for an html element.

Class Selector

Class selector uses class attributes of the elements to apply the sytle. All the elements having that class name will be formatted according to the defined rule
